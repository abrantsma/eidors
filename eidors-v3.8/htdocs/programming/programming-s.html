EIDORS: Information for developers

Getting the EIDORS Source

  • Windows:
    − Download and install Tortoise SVN
    − Right click in the directory you would like to download to, and type in https://eidors3d.svn.sourceforge.net/svnroot/eidors3d/trunk/
  • Unix / Linux: Type
    svn co https://eidors3d.svn.sourceforge.net/svnroot/eidors3d/trunk/

EIDORS Mailing List

  • All EIDORS questions should go to the [EIDORS-help] mailing list: eidors3d-help@lists.sourceforge.net.
  • To subscribe: https://lists.sourceforge.net/lists/listinfo/eidors3d-help"
  • The mailing list can be searched here: http://sourceforge.net/search/?group_id=100454&type_of_search=mlists

Becomming an EIDORS Developer

  1. Go the the EIDORS project page
  2. Click on the link for New User via SSL. Register your username
  3. Send a request for membership to the maintainers
